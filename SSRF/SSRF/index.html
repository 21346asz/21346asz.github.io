ctfshow

学习一些下面会用到的php函数

parse_url

解析url，返回关联数组，包含在url中出现的各种组成部分。数组的元素值不会url解码：

url: 要解析的URL

component: 指定 PHP_URL_SCHEME、PHP_URL_HOST、PHP_URL_PORT、PHP_URL_USER、PHP_URL_PASS、PHP_URL_PATH、PHP_URL_QUERY 或 PHP_URL_FRAGMENT 的其中一个来获取 URL 中指定的部分的 string。（当指定为 PHP_URL_PORT 时，返回值将是 int）。

返回值：

  1. 对不合格的URL, parse_url() 返回false。
  2. 如果省略了component参数，将返回关联数组array。
  3. 指定了component 返回与之相关的键的值。

curl_init

初始化新会话，返回cURL句柄，供curl_setopt()、curl_exec()和curl_close()函数使用。

url : 如果提供了该参数，curlopt_url 选项将被设置成这个值。也可以使用curl_setopt()函数手动设置这个值。

返回值: 成功时返回cURL句柄，错误时返回false。

curl_setopt

为cURL会话句柄设置选项。

handle 由curl_init() 返回的cURL句柄。

option 需要设置的CURLOPT_XXX选项。

value 将设置在option选项上的值。

返回值：成功时返回true，失败时返回false。

curl_exec

执行指定的cURL会话。

这个函数应该在初始化CURL会话并且设置所有选项后调用。

handle 由 curl_init() 返回的cURL 句柄。

返回值： 成功时返回true，失败时返回false。

curl_close

关闭cURL会话

$handle curl_init 返回的 cURL句柄。

返回值： 没有返回值。

gethostbyname

返回主机名对应的IPv4地址。

hostname 主机名

返回值： 成功时返回IPv4地址，失败时原封不动返回hostname字符串。

filter_var

使用特定的过滤器过滤一个变量

value 要过滤的内容。

filter 要应用的过滤器 ID。

option 一个关联数组，要过滤哪一部分。

成功时返回过滤后的数据，过滤失败返回false。
